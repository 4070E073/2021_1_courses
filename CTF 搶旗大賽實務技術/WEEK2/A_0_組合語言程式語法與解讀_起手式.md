# 使用gcc/g++產生組合語言[32bit與64bit不一樣!!]
```
產生預設的AT&T 語法的組合語言
    gcc -S func1.c -o func1.s

產生intel 語法的組合語言
    gcc -S -masm=intel func1.c -o func1_intel.s

gcc -S -masm=intel func1.c -o func1_intel.s -fno-asynchronous-unwind-tables
```
